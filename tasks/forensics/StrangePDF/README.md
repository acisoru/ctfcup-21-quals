## Описание

В PDF включён эксплоит работающий под старые версии Adobe reader. Эксплоит печатает флаг в консоль при работе.

## Решение

1. Открываем PDF Stream Dumper
2. Находим шеллкод
3. Выполняем его

## Флаг

`CUP{Y0u_h4v3_b33n_pwn3d!}`
